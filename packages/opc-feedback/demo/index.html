opc-feedback Component

opc-feedback component enables to collect the feedback from the user.

Supported Params

  1. url - URL of your hosted feedback spa. By default /feedback is the url.
  2. template - UI Template data input
    Javascript 
    document.getElementById('opc-feedback').template = {
                      "title": "How was your experience?.",
                      "subTitle": "It will help us to improve platform",
                      "summaryTitle": "Summary",
                      "summaryPlaceholder": "How can we do better?",
                      "errorTitle": "What is wrong?",
                      "experienceList": [{
                        "name": "Excellent",
                        "assetUrl": "./assets/happy.svg",
                        "errorList": []
                      },
                      {
                        "name": "Good",
                        "assetUrl": "./assets/good.svg",
                        "errorList": []
                      }]
                    };
    Template Properties Datatype
    title String
    subTitle String
    summaryTitle String
    summaryPlaceholder String
    errorTitle String
    confirmationTitle String
    confirmationSubTitle String
    experienceList Array

Themes

  1. opc-feedback in red(default) color
    2. HTML 
     <opc-feedback theme="red"></opc-feedback>
  2. opc-feedback in blue color
    3. HTML 
     <opc-feedback theme="blue"></opc-feedback>

Event Handling

  1. Obtaining the data from opc-feedback Component after the component action is initiated
    2. Javascript 
    document.querySelector('opc-feedback').addEventListener('opc-feedback:submit', (event) => console.log(event.detail.data));
  2. Open/Close the Feedback Modal universally
    3. Javascript 
    document.querySelector('opc-feedback').toggle();
